Learn how to make developers, creatives, content-makers and stakeholder work together.

Share preview

The most trivial way to share a page you are building is to open its preview and share the link with others. 
You can open the preview from the builder toolbar using the button.

Sharing direct link works only with public projects. To preview pages in privare repos users need to be collaborators.

Invite collaborators

To start collaborate with others you should first add them to your project. 
From the project page click on the button to open its preference, type the Github username for the new collaborator and click the "Add collaborator" button.

Propose changes
