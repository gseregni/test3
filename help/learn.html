Pingendo is a tool for developers and non-developers who wants to collaborate at building static web sites.

To satisfy the needs of everyone it comes in two versions 
Web that works in a browser as Github application - ideal for anyone 
Desktop that you need to download but works locally and offline - ideal developers that need need to integrates HTML and Sass with other framework or build tools 




Both versions features the same drag-and-drop builder.
The editor is simple to use but full-featured.
It support all Boostrap 4 components producing 100% vanilla code.
Theme support use Sass to be highly customizable with ease.
