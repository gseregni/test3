Pingendo for Github e' per la collaborazione




With Pingendo a user can browse Github repositories and edit HTML files in it. 

It works with both private and public repositories.

To access your own repo just replace github.com with gh.pingendo.com 
E.g.  github.com/twbs/bootstrap will be available at gh.pingendo.com/twbs/bootstrap






Plans will be per repository with unlimited collaborators.
It will work both with private and public repositories.
