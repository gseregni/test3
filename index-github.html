Pingendo for Github is a web app suitable for basic changes of text and images as well for static site developing.
It's integrated with Github to work on repositories and it saves changes from users as pull requests.
It enables clients, stakeholders, marketing and communication people to contribute easily and safely on the creation of static websites.

Visual editor

The editor works on static HTML and SASS files, allowing simple modification of contents for non developers as well as full authoring for developers.

Integrated with Github

Dead-simple navigation of repositories and proposal of changes as pull requests.

Efficient workflow for teams

Take everyone on Github for efficient collaboration of mixed teams. 
Non-developers can update and iterate issuing  pull request for developers

Safe

Developers manage users permissions and contributions with Github. Revisions are revertable at any time.  

A simple "no-CMS" solution

For who wants to keep things simple, Pingendo can be used to build, update and publish static web sites.

Open and versatile

Having Github as foundation, and generating plain HTML and SASS make Pingendo an open and versatile solution to build sites
