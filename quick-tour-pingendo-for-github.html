Pingendo for Github

Pingendo for Github it's a web app for dev and non-dev that needs to collaborate in a team building static website.
Web developers can share and get feedbacks from customers, marketing and content makers can build landing pages, 

#1 Login with Github

Pingendo integrates with GitHub to make your repos editable.
In order to start using it you'll need a Github account and to authorize Pingendo to access it.

#2 Setup your first repo

Once logged, your account page will show up.
Click on big green button to create an example repository, if you don't already have one.

#3
Work on repo

When watching a repo from PIngendo you'll get a simple view over HTML pages and project assets.
