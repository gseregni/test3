Pingendo

Pingendo for Github



Pingendo for Github it's a web app for dev and non-devs that needs to collaborate in a team building static website.
Web developers can share and get feedbacks from customers, marketing and content makers can build landing pages, 



Start building in few simple steps

Login with Github

Pingendo integrates with GitHub to make your repos editable.  
First time you login you'll need to authorize our app to access Github API.

Setup your first repo

Once logged your account page will show. Here are listed your repos on Github.
If you don't alredy have a repository containing Bootstrap pages, you can create an example one clicking on big green button

Work on/Manage repo

When watching a repo from PIngendo you'll get a simple view over HTML pages and project assets.

Share and build
