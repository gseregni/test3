Basic editing

For everyone

Double Click to edit text

Apply bold and more by the top bar

win ctrl z - mac cmd z.

ctrl C ctrl V
It work on text and on all elements

Add / replace images

Work on layouts

For devs & learners

Drag ready made sections or components

Release them before or after existing elements

Set element property

Work on source

For developers

Select an element to get at its HTML source

Dragging elements creates the related snippets in the HTML source. Every action on the visual editor updates the source.

The Saas editor simplifies adding custom rules. Saas  is compiled before being saved.
