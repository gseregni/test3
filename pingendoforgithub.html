GETTING STARTED IT'S EASY


Point your browser to the example repository we set up for you and authorize Pingendo on Github

github.pingendo.com/Pingendo/example-singlepage



Fork the example repository so you can write it.

    


Congrats, you are ready edit and push your prototypes.
Don't forget to add collaborators on repository to let others contributing.

Now that you are up and running, here are a few things you should know



Pingendo works with both private and public repositories.





Pingendo uses the same url scheme as  Github.  

To access your own repo just replace github.com with gh.pingendo.com 
E.g.  github.com/twbs/bootstrap  will be available at gh.pingendo.com/twbs/bootstrap


Plans will be per repository with unlimited collaborators.
It will work both with private and public repositories.
