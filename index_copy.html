Static site builder

Pingendo is a drag-and-drop Bootstrap builder integrated in a simplified Github interface. 
Created to enable developers and non-developers to build sites together from scratch without the overhhead of a CMS, shipping responsive HTML pages over wireframes and mockup.

Code quality first

Instant prototypes

Built on top of Github

Pingendo integrates with Github so your code is where it should be.

Up and Running in Seconds

All you need is a valid Github account. No installations required, it works on any Github repository.

Git Backed

Every project is hosted in a Github repository. So developers can manage users within Github and checkout the project locally.

Keep Your Work Safe

Changes from users are submitted as pull-requests and reversible at any time, so developers can spend time building, not worrying.

Publish For Free

Publish your projects in one click on Github Pages, the free hosting service kindly offered by Github.
